WordPressメディアライブラリの詳細管理と所有者変更機能を提供するプラグインです。

== Description ==

Kashiwazaki SEO Media Controlは、WordPressのメディアライブラリをより詳細に管理し、メディアファイルの所有者を効率的に変更できるプラグインです。

= 主な機能 =

* **詳細なメディア一覧表示**: アップロードされた全てのメディアファイルを詳細情報と共に一覧表示
* **所有者管理**: 各メディアファイルの現在の所有者を確認
* **個別所有者変更**: 個々のメディアファイルの所有者を簡単に変更
* **一括所有者変更**: 複数のメディアファイルの所有者を一度に変更
* **豊富な情報表示**: ファイル名、タイトル、タイプ、サイズ、アップロード日、所有者などを表示
* **フィルター機能**: メディアタイプや所有者でフィルタリング可能
* **レスポンシブデザイン**: モバイルデバイスでも使いやすいインターフェース

= 使用場面 =

* マルチユーザーサイトでのメディア管理
* サイト運営者変更時のメディア所有権移譲
* コンテンツ管理の効率化
* SEO最適化のためのメディア整理

== Installation ==

1. プラグインファイルを `/wp-content/plugins/kashiwazaki-seo-media-control` ディレクトリにアップロード
2. WordPress管理画面の「プラグイン」メニューからプラグインを有効化
3. 管理画面のサイドメニューに「Kashiwazaki SEO Media Control」が追加されます
4. 「Kashiwazaki SEO Media Control」をクリックしてメディア管理画面にアクセス

= 最低要件 =

* WordPress 5.0以上
* PHP 7.4以上
* 管理者権限（所有者変更機能使用時）

== Frequently Asked Questions ==

= プラグインを有効化後、どこからアクセスできますか？ =

WordPress管理画面の左側メニューに「Kashiwazaki SEO Media Control」が追加されます。クリックするとメディア管理画面にアクセスできます。

= 所有者を変更するには管理者権限が必要ですか？ =

はい、セキュリティ上の理由から、メディアファイルの所有者を変更するには管理者権限（edit_users権限）が必要です。

= 一括変更はどのように行いますか？ =

1. メディア一覧でチェックボックスを使用して変更したいメディアを選択
2. 「所有者を選択」ドロップダウンから新しい所有者を選択
3. 「選択したメディアの所有者を一括変更」ボタンをクリック
4. 確認ダイアログで「OK」をクリック

= このプラグインはマルチサイトに対応していますか？ =

現在のバージョンはシングルサイト専用です。マルチサイト対応は今後のアップデートで検討予定です。

= メディアファイルが削除されることはありますか？ =

いいえ、このプラグインはメディアファイルの所有者情報のみを変更し、ファイル自体を削除することはありません。

== Privacy Policy ==

このプラグインは以下の情報を処理します：

* **メディアファイル情報**: ファイル名、タイプ、サイズ、アップロード日
* **ユーザー情報**: 所有者の表示名とユーザーID
* **変更履歴**: 所有者変更の記録（WordPress標準のログ機能を使用）

これらの情報は全てWordPress内部で処理され、外部のサーバーに送信されることはありません。
